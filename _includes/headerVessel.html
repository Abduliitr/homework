Shell Thickness Calculation

Useful DATA:

  • Class I:
    1. J=1.0, for fully radiographed joints.
    2. J=0.9, for single welded butt joints with backing strips.
  • Class II:
    1. J=0.85, for double welded butt joints with full penetration.
    2. J=0.80, for single welded butt joints with backing strips.
  • Class III:
    1. J=0.70, for double welded butt joints with full penetartion.
    2. J=0.65, for single welded butt joints with backing strip.
    3. J=0.60, for single welded butt joints with backing strip which may not remain in place.
    4. J=0.55, for single welded butt joints without backing strip
    5. J=0.50, for single full lap joints for circumferential seams only.
